💾 Exporter la sauvegarde

Télécharge un fichier JSON contenant toutes vos données (clients, ventes, recouvrements, stock, etc.).
Conservez ce fichier en lieu sûr — il vous permettra de tout restaurer.

✅ Ne contient jamais vos mots de passe ni vos identifiants de connexion BD.
✅ Peut être importé sur n'importe quelle instance de l'application.
Restaurer une sauvegarde

La restauration s'effectue depuis la page Sauvegarde BD (menu lateral) en téléchargeant un fichier .sql.

Pour restaurer depuis un fichier .sql :
1. Télécharge une sauvegarde depuis le menu Sauvegarde BD
2. Ouvre un terminal dans le dossier du projet
3. Exécute :
mysql -u root -p positive_distribution < backup_XXXX.sql
